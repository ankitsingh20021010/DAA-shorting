Huffman Coding

Definition

Huffman Coding is a lossless data compression algorithm. It assigns shorter binary codes to characters with higher frequency and longer codes to characters with lower frequency.

Steps of Huffman Coding

  1. Count frequency of each character.
  2. Create a node for each character.
  3. Build a priority queue (min heap).
  4. Take two nodes with lowest frequency.
  5. Create a new node with sum of their frequencies.
  6. Repeat until only one node remains (Huffman Tree).
  7. Assign 0 to left edge and 1 to right edge.

Example

Character Frequency Huffman Code
A 3 101
B 2 100
C 6 11
D 8 0

Encoded Output Example

If message is:

ABCD

Encoded binary:

101 100 11 0

Advantages

  • Reduces file size
  • Lossless compression
  • Efficient for data transmission
